About Creature Feature

What is this?

Creature Feature is a task management app created by Mars-Morgan Anthony and Fallon Katz for their Mobile Application Development class. (We wanted to do something simple for the project but we had a little too much fun.)

How it works

  • Creature designs are displayed as animated GIFs with accessories (Fallon drew them!)
  • Achievements are fetched from a REST-style GitHub Pages endpoint
  • Tasks are stored locally using SQLite
